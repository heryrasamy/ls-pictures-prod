# LS Pictures Productions – Maquette "pour voir"

Ce dépôt contient une maquette HTML cliquable destinée à valider le parcours client
avant toute intégration ou modification sur le site Wix existant.

## Objectif
- Clarifier le parcours utilisateur
- Valider la logique : découverte → demande de devis → devis → paiement
- Sécuriser la suite du projet sans toucher au site actuel

## Contenu
La maquette comprend :
- Une page d’accueil
- Une page services
- Une page service type
- Une page de demande de devis
- Une page de confirmation

## Remarque
Cette version est une base de travail.
Aucun paiement réel, aucune automatisation active.
